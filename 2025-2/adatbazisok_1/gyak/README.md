# Adatbázisok 1 gyakorlat - ZH információk

## Gyakorlati követelmények (2025/2026. II. félév)

- Személyes jelenléti oktatás van.
- A gyakorlati jegy feltétele:
  - aktív órai részvétel,
  - kötelező órai feladatok beadása,
  - 2 db ZH teljesítése.
- Minimum 7 kötelező órai beadandó szükséges (max. 3 hiányzás).
- Aki nem teljesíti a minimum beadandót, nem kaphat gyakorlati jegyet.

## Általános ZH szabályok

- Két ZH van: a 6. és a 12. gyakorlaton.
- Mindkét ZH-n legalább 2-est kell elérni.
- Igazolatlan hiányzás ZH-n: 1-es.
- Csak az egyik ZH javítható; javítás után a régi és új pont átlaga számít.
- Ha javítás után is bármelyik ZH 1-es, a gyakorlati jegy elégtelen.
- ZH alatt csak a labor gépek használhatók.
- Megengedett segédletek:
  - Oracle Database SQL Language Reference
  - Oracle Database PL/SQL Language Reference (ZH2-höz)
- Minden más segédeszköz, csevegő, közösségi oldal, mobil használata tilos.
- A ZH megsértése elégtelen érdemjegyet von maga után.

## ZH1 és ZH2 részletek

<details>
<summary><strong>ZH1 (I. ZH) - követelmények és tudnivalók</strong></summary>
- Felépítés: relációs algebra (50%) + SQL lekérdezések (50%).
- Relációs algebra résznél csak alap műveletek használhatók (pl. vetítés, kiválasztás, átnevezés, természetes összekapcsolás, unió, különbség; továbbá direkt szorzat és metszet).
- Relációs algebra résznél segédlet nem használható.
- Minimum követelmény: relációs algebra részből legalább 40% (a teljes ZH 20%-a).
- SQL rész Oracle környezetben, minden helyes megoldás elfogadható.
- A megoldásokat folyamatosan érdemes menteni a Canvasba; végleges megoldásnál outputot (vagy hiba esetén hibaüzenetet) is be kell másolni.
- A lekérdezéseknek általánosan helyesnek kell lenniük, nem csak aktuális adatokra.

</details>

<details>
<summary><strong>ZH2 (II. ZH) - követelmények és tudnivalók</strong></summary>

- Felépítés: SQL DDL/DML (50%) + PL/SQL feladatok (50%).
- SQL rész: DML, DDL (create table, constraint, view), hierarchikus lekérdezések (CONNECT BY).
- PL/SQL rész: SELECT INTO, implicit/explicit kurzorok, tárolt eljárások/függvények, kivételkezelés.
- Minimum követelmény: PL/SQL részből legalább 40% (a teljes ZH 20%-a).
- A megoldásokat folyamatosan érdemes menteni a Canvasba; végleges megoldásnál outputot (vagy hiba esetén hibaüzenetet) is be kell másolni.

</details>
