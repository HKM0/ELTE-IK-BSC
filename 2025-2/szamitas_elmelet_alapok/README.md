# Számításelmélet Alapok

## Vizsgakövetelmények

### Zárthelyik
- **2 ZH** minimum **(15-15 pont)**
- Kettes érdemjegyhez **minimum 40%** teljesítése

### Értékelés
| Érdemjegy | Szükséges pontszám |
|-----------|-------------------|
| 2 (elégséges) | min 40% |
| 3 (közepes) | 53 pont |
| 4 (jó) | 67 pont |
| 5 (jeles) | 80 pont |

## Pótlási lehetőségek

### Pótzárthelyi
- Javítható **bármelyik** a két ZH közül
- **Mindkettő** javítására van lehetőség

### Gyakorlati utóvizsga
- Ha a ZH-k nem sikerülnek, van gyakorlati utóvizsga
- **40% minimum** a teljes tananyagból
